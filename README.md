# EmpTrack: Employee Insight System

## GUI-Based Employee Management System using Python, SQLite, PyQt5 and Speech Recognition

EmpTrack is a Human-Computer Interaction (HCI) focused Employee Management System developed using Python, PyQt5, and SQLite.

The system allows administrators to efficiently manage employee records through a modern graphical interface while integrating voice-based interaction features to improve accessibility and user experience.

Each employee record contains:

* Employee ID
* Name
* Last Name
* Age
* Position
* Salary
* Email Address

The system supports employee management, analytics, data visualization, email communication, profile management, and voice-enabled search functionality.

This project was developed as part of the Human-Computer Interaction (HCI) course.

To enhance usability and accessibility, the system integrates voice recognition capabilities using PyAudio and Google Speech Recognition API. Users can interact with the application through speech-based input, reducing manual effort and improving overall user experience.
